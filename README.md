# Personal Portfolio Website 🚀

A modern full-stack personal portfolio website built using the MERN Stack to showcase projects, skills, resume, and contact information.

## 🌟 Features

* Responsive modern UI
* Hero section with introduction
* About section
* Skills section
* Projects showcase
* Contact form integration
* Resume download button
* GitHub & LinkedIn links
* Backend API using Node.js & Express.js
* Mobile responsive design

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/divyakakumanu012-cmd/portfolio_project.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
npm install
npx nodemon server.js
```
